那张Windows XP的经典“Bliss”壁纸又在现实中短暂出现了 by cnbeta
------
<div style="margin-top:10px" class="content" id="artibody"><p>近日，一位 Reddit 用户实地探访了著名的 Windows XP 默认壁纸“Bliss”的拍摄地点，并意外发现，这片位于美国加州纳帕谷的山丘，在特定天气和季节条件下，竟再次呈现出与当年几乎一致的景象，引发大批网友怀旧与热议。</p><div class="article-global"></div><p>这名网友与朋友一同前往“Bliss”所在地，在停下车拍照时，捕捉到了一幅与经典壁纸极为相似的画面：碧绿的草坡起伏延展、远处山丘线条柔和，只是天空中的云层形态略有差异。他表示，这样的景象“极其罕见”，因为这座山丘在大多数时候要么被葡萄藤覆盖，要么是一片干枯的黄色草地。有其他网友补充称，近期加州的湿润天气，使得当地植被恢复了生机，让这片山丘得以在现实中再次“演绎”出 XP 壁纸般的清新绿意，只是云层不再如当年的照片那般恰到好处。</p><p style="text-align: center;"><img src="https://static.cnbetacdn.com/article/2026/0221/e1af66356de6835.webp"></p><p>不少网友在看到新旧照片对比后，干脆将这张新拍的山丘照片设置为自己的 Windows 11 壁纸，以此向经典致敬。而居住在附近的居民则表示，打算趁着山丘暂时没有葡萄种植行列或病虫害的时期，再去亲眼看一看这个现实中的“Bliss”。</p><p>“Bliss”的原始照片拍摄于 1996 年，作者是当时供职于《国家地理》的摄影师 Charles O'Rear。这张照片后来作为库存图片，被图片社 Corbis 收录并对外授权。2000 年，微软购买了该照片的使用权，并在随后发布的 Windows XP 中，将其设为默认桌面壁纸。基于 NT 架构的 Windows XP 在前五年间售出逾 4 亿份拷贝，这意味着全球有数以亿计的用户——乃至更多通过媒体报道接触到 XP 界面的公众——都曾至少看过这张壁纸一次。</p><p><a href="https://img.3dmgame.com/uploads/images/news/20251013/1760352198_120045_jpg_r.jpg" target="_blank"><img src="https://img.3dmgame.com/uploads/images/news/20251013/1760352198_120045_jpg_r.jpg"></a></p><p>O'Rear 当年使用的是 Mamiya RZ67 中画幅胶片相机，在前往探望当时的女友途中，顺手记录下了眼前这片丘陵风光。由于照片色彩明快、构图简洁，有关它是否经过 Photoshop 大幅润饰的猜测长期不绝于耳，但 O'Rear 一直坚持称，这是一张未被后期“动过刀”的自然作品。相比之下，这位 Reddit 网友如今拍下的“新版 Bliss”显然是用智能手机完成的，画面质感也自然带有明显的数码时代特征。</p><p>在 Windows XP 发布 25 年之后，微软在为新一代操作系统挑选壁纸时，已经走上了截然不同的路线。Windows 11 中自带的壁纸多为高度修饰的合成图像或由人工智能生成的画面，真正来自自然风景、并以纯粹景色取胜的照片相对有限。也正因如此，当人们再次看到现实世界中的“Bliss”与记忆中的 XP 桌面重叠时，不免产生一种时代错位般的感慨：从那片简单的绿坡到如今光怪陆离的数字图像世界，“一切从什么时候开始变了？”</p></div>
