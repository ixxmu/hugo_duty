Google Gemini最“小”的一次迭代姚顺宇为何高呼“不可阻挡”？ by cnbeta
------
<div style="margin-top:10px" class="content" id="artibody"><p>人们还在津津乐道OpenAI和Anthropic两家公司的老板拒绝牵手，以至于握拳高举的滑稽画面，另一边，Google一个反手迭代了模型。</p><div class="article-global"></div><p><img src="https://static.cnbetacdn.com/article/2026/0221/1ed3ae94eb6b64a.webp"><br></p><p>而且这次迭代，颇像是狼披上了羊皮——从模型后缀的数字序号看，这是Google最“小”的一次迭代，之前都是从Gemini 2.0到Gemini 2.5这样的迭代，这次从Gemini 3.0到Gemini 3.1Pro Preview。</p><p><img src="https://static.cnbetacdn.com/article/2026/0221/2eb027d9e14a26d.webp"><br></p><p><strong>但是这“.1”的迭代，进步得却不是一星半点。</strong></p><p>GoogleCEO皮查伊（Sundar Pichai）表示，新代模型非常擅长处理“超级复杂的任务”。例如将复杂概念可视化、将数据综合成单一视图或者将创意项目变为现实。</p><p><img src="https://static.cnbetacdn.com/article/2026/0221/610d336e061230e.webp"><br></p><p>姚顺宇也专门在X上发帖为Gemini 3.1 Pro Preview振臂，盛赞：</p><p><strong>“Gemini不仅仅是一个好模型，更好的模型正在以不可阻挡之势到来。”</strong></p><p><img src="https://static.cnbetacdn.com/article/2026/0221/a0c83cbab224f52.webp"><br></p><p>需要注意的是，大约一周前，Google推出了“专用推理模式”Gemini 3 Deep Think，专为科学、研究、工程等复杂、开放式问题设计。</p><p>Demini 3 Deep Think是姚顺宇从Anthropic跳槽到GoogleDeepMind之后参与的第一个项目。</p><p>而如今的Gemini 3.1 Pro Preview和Gemini 3 Deep Think有千丝万缕的关系，官方表示“直接构建在Gemini 3 Deep Think的经验和技术之上”。相当于把Deep Think的核心推理提升技术“下放”到更广泛可用的Pro模型中。</p><p><strong>01</strong></p><p><strong>Gemini 3.1 Pro Preview能做什么</strong></p><p>既然这次新代模型的突出能力在处理“超级复杂”的任务，那么普通的对话放一边，Google的官方博文中，重点给出了几种示例秀肌肉。</p><p><strong>第一，通过简单的提示词，创建SVG动画。</strong></p><p>这个功能前代也有，但是对比之下进步明显。</p><p>比如提示词“生成一个SVG，描绘一只变色龙静静地坐在树枝上。让变色龙的眼睛跟随用户的鼠标光标在屏幕上移动。”</p><p>Gemini 3 Pro生成的动画背景是单一的白色，变色龙也看起来很呆板，甚至一侧有两只眼睛。</p><p>Gemini 3.1 Pro生成的动画则有丰富的“深绿色丛林”背景，变色龙身体有黄色斑纹和圆点装饰，眼睛立体，腿部姿势自然弯曲。</p><p><img src="https://static.cnbetacdn.com/article/2026/0221/178a8aafd8bfe79.gif"><br></p><p>再比如提示词“生成一个滑动切换开关的SVG，当鼠标悬停在太阳图标上时，将其变成发光的月亮，同时背景从明亮平滑渐变到黑暗。采用干净的扁平UI风格。”</p><p>Gemini 3 Pro给出的动画虽然完成了任务，图标可以随鼠标变化，但是主图标单一，是一个缺角的圆形图案，用黄色代表白天，白色代表夜间。</p><p>Gemini 3.1 Pro生成的动画要复杂得多，白天是黄日白云，夜间是月牙繁星，两套图标丝滑变换。</p><p><img src="https://static.cnbetacdn.com/article/2026/0221/bb5d25f8c024575.gif"><br></p><p>总而言之，Gemini 3 Pro做的动画，让人想起多年前那个“学了三年动画”的梗。</p><p><img src="https://static.cnbetacdn.com/article/2026/0221/9fed3346f92728f.gif"><br></p><p>而Gemini 3.1 Pro交付的SVG动画，已经达到了可以直接使用的效果。</p><p>第二，构建工程级别系统。</p><p>Gemini 3.1 Pro已经可以根据一段高复杂度的自然语言指令，直接生成一个集3D渲染、实时太阳星历计算、API异步拉取和物理光效于一体的完整交互系统，而不是简单页面demo。</p><p>Google给出的示例中，用户给出文字指令，Gemini 3.1 Pro生成了构建一个高保真、可交互的3D国际空间站（ISS）轨道追踪器。使用高分辨率的Blue Marble纹理贴图渲染一个精细的3D地球模型。</p><p><img src="https://static.cnbetacdn.com/article/2026/0221/1ddbec2cae0f941.gif"><br></p><p><strong>第三，生成交互式创意系统。</strong></p><p>在另一个示例中，Google展示了Gemini 3.1 Pro编写的一个复杂的3D椋鸟群舞（murmuration）模拟。</p><p><img src="https://static.cnbetacdn.com/article/2026/0221/f37210140ffb6da.gif"><br></p><p>它不仅生成视觉代码，还构建了一个沉浸式体验，用户可以通过手部追踪操控鸟群，并聆听根据鸟群运动变化的生成式配乐。</p><p>对于研究人员和设计师而言，这提供了一种强有力的方式来原型化感官丰富的界面。</p><p><strong>第四，将文学主题转化为可运行的代码。</strong></p><p>这个示例可能是普通人也最容易get到厉害之处的一个。</p><p><img src="https://static.cnbetacdn.com/article/2026/0221/51b5c9760d31be8.gif"><br></p><p>当被要求为艾米莉·勃朗特（Emily Brontë）的《呼啸山庄》（Wuthering Heights）打造一个现代个人作品集网站时，该模型并未简单地总结文本内容，而是基于小说的氛围与情绪进行推理，设计出一个简洁、当代感十足的界面，创建出一个捕捉主人公精神内核的网站。</p><p>抽象推理，这种能力的含金量不用多说。</p><p><strong>02</strong></p><p><strong>到底有多强？</strong></p><p>新代模型免不了过一下刷榜这一步。</p><p>而“.1”的升级，获得了动辄倍数跃升的成绩。</p><p>根据Google官方博文放出的测试结果，</p><p>在ARC-AGI-2基准测试中，3.1 Pro的验证得分达到了77.1%。比3 Pro的推理性能提升了一倍以上。</p><p>这也符合3.1 Pro的示例，因为这个测试评估的是模型解决全新逻辑模式的能力。用人话说就是，抽象推理解谜的能力。</p><p><img src="https://static.cnbetacdn.com/article/2026/0221/5c840a10b3bd601.webp"><br></p><p>此外，在GPQA Diamond（科学知识测试）中，3.1 Pro得分94.3%；智能体类基准MCP Atlas上，得分69.2%；真实网络浏览与信息整合能力的基准BrowseComp上，得分85.9%。</p><p>这些成绩，均超过了Anthropic的Sonnet 4.6、Opus 4.6，以及OpenAI的GPT-5.2与GPT-5.3-Codex。</p><p>GoogleGemini 3.1 Pro这次在ARC抽象推理和BrowseComp搜索任务上显著拉开差距，显示出明显的Agent倾向，而非单纯知识模型。</p><p>此外，专门做大模型基准测试与对比分析的第三方评测机构也发布了相关测试结果，大赞Gemini 3.1 Pro在构成Artificial Analysis Intelligence Index的10项评测中领先6项，相较于Gemini 3 Pro Preview在多项能力上显著提升，尤其是在推理与知识、代码能力以及降低幻觉方面进步最大。</p><p><img src="https://static.cnbetacdn.com/article/2026/0221/5d4574cc6e9aef8.webp"><br></p><p>而且，Gemini 3.1 Pro Preview保持较高的token效率。</p><p>运行完整Artificial Analysis Intelligence Index约需5700万token（比Gemini 3 Pro Preview多100万）。</p><p><strong>这一token使用量低于其他在最大推理模式下运行的前沿模型，如Opus 4.6（max）和GPT-5.2（xhigh）。</strong></p><p>结合更低的单token定价，Gemini 3.1 Pro Preview在前沿模型中具有成本优势，运行完整Intelligence Index的成本不到Opus 4.6（max）的一半，不过仍约为领先开源模型GLM-5的两倍。</p><p><strong>03</strong></p><p><strong>能力翻倍价格不变</strong></p><p>Google官方API定价显示，Gemini 3 Pro/3.1 Pro Preview的收费结构是按token计费的：</p><p>小于200k tokens时，每百万token输入约2美元，输出价格为4美元。大于200k tokens时，每百万token输入4美元，输出18美元。</p><p>在上下文缓存方面，根据提示词规模，每百万 token 收取 0.20 至 0.40 美元，外加每小时每百万 token 4.50 美元的存储费。</p><p><strong>这个价格，整体上和Gemini自己的上一代3 Pro一致，但要是和 Anthropic Opus 系列比起来，还是相对便宜的。 Opus这类模型输入/输出单价能在$5/$25左右。</strong></p><p>尤其是结合当下它在模型能力上相当突出，这个价格更显得极具竞争力。</p><p>不要忘了，Google这次发布的只是“预览（Preview）”，Google很快就会推出正式版。而“.1”的迭代，Google也是在暗示其只是小秀肌肉。</p><p>目前，开发者可以在AI Studio、Gemini API、Gemini CLI、智能体开发平台Google Antigravity以及Android Studio使用3.1 Pro；企业用户则可以在Vertex AI和Gemini Enterprise使用；普通用户在Gemini应用和NotebookLM都能用上，不过后者仅限Pro和Ultra订阅用户。</p><p><strong>各个社区里已经有不少人迫不及待上手操作，真的就和Google的演示一样，手搓了不少惊艳的玩意。</strong></p><p>有人用Gemini 3.1 Pro 生成了一个可交互的 3D 机械级汽车悬架系统模拟器，包含真实几何结构、连杆约束与实时转向和行程计算，相当于把机械工程建模、物理逻辑和3D可视化一次性写成可运行工具，接近工程级原型能力。</p><p><img src="https://static.cnbetacdn.com/article/2026/0221/a0c6c5cede6ebfd.gif"><br></p><p>有人用3.1 Pro制作“捉鬼猎人走过一栋闹鬼的房子”的循环动画，惊呼“Gemini没有开玩笑”。</p><p><img src="https://static.cnbetacdn.com/article/2026/0221/6f9b086aaa9a607.gif"><br></p><p>总之，Google这回是真憋了个大招。</p><p>小小一个“.1”，却把推理和代码能力拉得飞起，定价还这么稳。</p><p>社区里满溢的手搓demo的热情，也证明了它的能力与实用性。</p><p>AI圈越来越现实了。模型再强，终究得看账单值不值。企业开始精打细算每个token的回报，开发者也得掂量掂量性价比。Google这一步，不仅是抢回王座，更是把竞争推向“谁更会过日子”的新阶段。</p><p>且看接下来，捏着拳头的Anthropic和OpenAI，以及xAI、Meta、微软等一众竞争对手该如何应对吧。</p></div>
